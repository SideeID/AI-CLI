# Side ID AI CLI

## Deskripsi
Aplikasi Command Line Interface (CLI) untuk berinteraksi dengan LLM.

## Prasyarat
- Python 3.8+
- pip

## Instalasi
1. Clone repositori
2. Buat virtual environment
3. Aktifkan virtual environment
4. Install dependencies: `pip install -r requirements.txt`

## Penggunaan
```bash
python side_ai.py
